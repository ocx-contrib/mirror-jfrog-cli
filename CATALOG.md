# JFrog CLI

JFrog CLI is a command-line client that provides a simple interface to automate
access to the JFrog Platform — Artifactory, Xray, Distribution, and more. It
streamlines uploads, downloads, builds, scans, and CI/CD integrations from a
single `jf` binary.

## What's included

- **jf** — the JFrog CLI client (build, scan, upload/download, config, and Platform automation)
